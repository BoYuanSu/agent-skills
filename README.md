# Agent Skills

這個專案是受到 [obsidian 與 AI 工作流整合](https://www.youtube.com/watch?v=ZC8AI_jFWnU&t=3s) 的啟發，旨在建立一個以知識為基礎的技能庫，透過 AI 建立 Agent Skills 的專案。

並且透過這個專案累積開發經驗並更新相關 skills，讓這些 skills 應用專案時，透過更新此專案 skills 達到統一管理散落在各專案的 skill 資訊，讓開發更加有效率。

## Installation

```bash
npx skills@latest add boyuansu/agent-skills
```
